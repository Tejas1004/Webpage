What's the Difference?

HTML

Hyper Text Markup Language

 Create the structure

  • Controls the layout of the content
  • Provides structure of the webpage design
  • The fundamental building block of any webpage

CSS

Cascading Style Sheet

 Stylize the website

  • Applies style to the webpage elements
  • Target various screen sizes to make webpage responsive
  • Primarily handles the "look and feel" of a webpage

Javascript

 Increase interactive

  • Adds interactivity to a webpage
  • Handles complex functions and feautres
  • Programatic code which enhances functionality
